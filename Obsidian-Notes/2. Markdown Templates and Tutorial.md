# Section 2: Text format
---
#### 2.1. Bold text

**Bold text** and other __bold text__ -> Hotkey: Ctrl+B

#### 2.2. Italic text

*Italic text* and other _italic text_-> Hotkey: Ctrl+I

#### 2.3. Bold and italic mixed

***Bold and italic text*** and other ___Bold and italic text___ -> Hotkey: Ctrl+B & Ctrl+I

#### 2.4. Highlighted text

==Highlighted text== -> Hotkey: Ctrl + Shift + S ==(custom)==

![[Pasted image 20260319231143.png]]

#### 2.5. Crossed out text

~~Crossed out text~~ -> Hotkey: Ctrl + Shif + C ==(custom)==

![[Pasted image 20260319231229.png]]

#### 2.6. Underlined text

The Markdown format doesn't allows underlined text by default. This is to avoid confusion with hyperlinks. However, we can achieve underlining using HTML.

<u>Hello robot, are you a 1 or a 0?</u>

#### 2.7. About spaces

You can't have more than one blank space between paragraphs. Even if you add more than one in editor mode, you'll see that in reading mode they're removed, leaving only one.
